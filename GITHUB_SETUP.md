## üöÄ Step-by-Step Instructions

### Step 1: Go to Walmart GitHub
1. Open your browser
2. Go to one of these URLs (depending on your access):
   - **https://github.wmt.com** (most common)
   - **https://github.walmart.com** (alternative)
   - **Or your internal GitHub instance**
3. Log in with your **Walmart credentials**

### Step 2: Create a New Repository

**Look for one of these:**
- A **"+"** button (usually top right)
- A **"New"** button (usually top left)
- A **"Create repository"** option

**Click it and select "New repository"**

### Step 3: Configure Repository Settings

On the "Create a new repository" page, fill in:

| Field | Value |
|-------|-------|
| **Repository name** | `fpa-lpa-dashboard` |
| **Description** | `FPA/LPA Performance Dashboard for DC 6084` (optional) |
| **Visibility** | **Private** (important - internal only) |
| **Initialize this repository with** | ‚ùå Leave UNCHECKED (we have code already) |
| **Add .gitignore** | Leave empty |
| **Add a license** | Leave empty |

### Step 4: Create the Repository

1. Click **"Create repository"** (blue button)
2. Wait a few seconds for GitHub to create the repo
3. You'll see a page with setup instructions ("Quick setup")

### Step 5: Get Your Repository URL

**THIS IS THE IMPORTANT PART:**

1. On the setup page, look for a **green "Code" button** (or "Clone" button)
2. Click it
3. Select **"HTTPS"** (should be default)
4. **Copy the URL**

It will look like one of these:
```
https://github.wmt.com/your-username/fpa-lpa-dashboard.git
https://github.walmart.com/your-username/fpa-lpa-dashboard.git
```

**COPY THIS EXACT URL** - you'll need it in the next step!

## üÜò Troubleshooting

### "I can't find my GitHub instance"
- Try: https://github.wmt.com
- Or ask your IT team for your internal GitHub URL
- Your team might have a specific GitHub instance

### "I can't log in"
- Use your Walmart email and password
- Make sure you have GitHub access (contact IT if not)
- Try clearing browser cookies and logging in again

### "I don't see the Code button"
- Make sure you're on your new repository's main page
- The "Code" button is usually green and in the top right area
- Refresh the page if you don't see it

### "The URL looks different"
- GitHub URLs should look like:
  - `https://github.wmt.com/username/fpa-lpa-dashboard.git` ‚úÖ
  - `https://github.walmart.com/username/fpa-lpa-dashboard.git` ‚úÖ
  - `https://your-github-instance/username/fpa-lpa-dashboard.git` ‚úÖ
- Just copy whatever URL GitHub shows you

### "I need to change to Private"
1. Go to **Settings** (top menu)
2. Scroll down to **Repository visibility**
3. Select **Private**
4. Click **Update visibility**
